# 🫁 Détection de Pneumonie par Intelligence Artificielle

Cette application web permet d'analyser des radiographies thoraciques pour détecter automatiquement la présence de pneumonie. Le système repose sur un modèle de Deep Learning (TensorFlow/Keras) intégré dans une architecture backend robuste pour offrir une interface simple aux professionnels de santé.

## 🛠️ Technologies Utilisées
* **Backend :** Python, Django
* **Base de données :** MySQL (via XAMPP/MariaDB)
* **Intelligence Artificielle :** TensorFlow, Keras (Modèle `.h5`)
* **Frontend :** HTML5, CSS3, Bootstrap 5
